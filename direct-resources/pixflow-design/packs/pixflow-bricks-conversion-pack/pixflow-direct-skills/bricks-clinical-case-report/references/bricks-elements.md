# Bricks Clinical Case Report Bricks Element Rules

- Build the final output as Bricks JSON, not HTML, React, SVG-only mockups, or native app code.
- Use stable short BEM global classes such as `.clinical-case-report__panel` and `.clinical-case-report__item`.
- Use IDs or custom attributes only when the section needs a stable target for future interaction code.
- Do not output unsupported Bricks element names.
