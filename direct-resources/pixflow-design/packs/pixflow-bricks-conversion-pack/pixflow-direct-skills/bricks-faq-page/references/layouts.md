# Bricks FAQ Page Layout Guidance

Use the layouts in layouts.json as the approved skeleton pool. Pick section skeletons before writing copy. Do not use the same centered heading plus equal-card grid repeatedly.

Recommended rhythm:

- `faq_hero`: use `hero_split_visual` with content specific to faq-page.
- `faq_category_stack`: use `faq_or_objection_stack` with content specific to faq-page.
- `faq_featured_answers`: use `bento_features` with content specific to faq-page.
- `faq_support_split`: use `story_split` with content specific to faq-page.
- `faq_final_cta`: use `faq_or_objection_stack` with content specific to faq-page.
