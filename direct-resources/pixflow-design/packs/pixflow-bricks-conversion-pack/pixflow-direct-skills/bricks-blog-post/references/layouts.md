# Bricks Blog Post Layout Guidance

Use the layouts in layouts.json as the approved skeleton pool. Pick section skeletons before writing copy. Do not use the same centered heading plus equal-card grid repeatedly.

Recommended rhythm:

- `article_hero`: use `magazine_style_hero` with content specific to blog-post.
- `article_intro_callout`: use `magazine_style_hero` with content specific to blog-post.
- `article_content_split`: use `magazine_style_hero` with content specific to blog-post.
- `article_support_cards`: use `magazine_style_hero` with content specific to blog-post.
- `article_final_cta`: use `magazine_style_hero` with content specific to blog-post.
