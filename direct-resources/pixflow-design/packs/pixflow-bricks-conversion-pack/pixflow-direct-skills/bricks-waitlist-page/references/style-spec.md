# Bricks Waitlist Style Spec Contract

## Token Rules

- Use `var(--background)`, `var(--foreground)`, `var(--heading)`, and `var(--text)`.
- Signup panels use `var(--radius--surface)` and `var(--border)`.
- Buttons use `var(--radius--btn)` and `var(--border--btn)`.

## Layout Style Rules

- Keep composition compact and conversion-focused.
- Use accent for signup action and one launch/status cue.
- Avoid excessive decorative polish.
