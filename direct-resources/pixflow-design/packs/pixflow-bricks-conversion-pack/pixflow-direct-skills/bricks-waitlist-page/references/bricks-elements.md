# Bricks Waitlist Element Contract

## Allowed Elements

- `section`, `container`, `block`, `div`
- `heading`, `text-basic`, `button`

## Structure

- Signup form surfaces are editable blocks and text fields; do not output live form scripts.
- Product previews use editable UI blocks when no real screenshot is supplied.
- FAQ and CTA use simple Bricks text/button elements.

## Bricks Rules

- Use BEM blocks such as `hero`, `promise`, `timeline`, `benefits`, `faq`, and `signup`.
- Do not output raw form HTML or embed scripts.
