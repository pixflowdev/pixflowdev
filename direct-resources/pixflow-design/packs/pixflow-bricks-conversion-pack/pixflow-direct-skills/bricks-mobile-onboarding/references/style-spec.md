# Bricks Mobile Onboarding Style Spec Contract

Use the active Pixflow design system as final authority. In Pixflow Framework mode, sections use ar(--background), cards and panels use ar(--foreground), borders use ar(--border), buttons/fields use ar(--radius--btn) and ar(--border--btn), and surfaces/media use ar(--radius--surface) and ar(--border--surface).

Keep visual quality high through hierarchy, spacing, composition, and useful supporting panels. Do not add raw hex colours, fallback var chains, decorative gradients, or stock-photo substitutions when the prompt asks for editable interface or product structures.
