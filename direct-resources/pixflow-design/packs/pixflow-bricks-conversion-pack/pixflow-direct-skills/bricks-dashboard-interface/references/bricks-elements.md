# Bricks Dashboard Interface Element Contract

## Allowed Elements

- `section`, `container`, `block`, `div`
- `heading`, `text-basic`, `button`

## Structure

- UI mockups are nested Bricks blocks: browser frame, nav rail, toolbar, metric cards, chart blocks, table rows, and settings panels.
- Use text-basic for labels and sample values.
- Use buttons only for real CTAs or interface controls.

## Bricks Rules

- Use BEM blocks such as `hero`, `dashboard`, `workflow`, `features`, `faq`, and `cta`.
- Do not output canvas, script, or unsupported chart embeds.
