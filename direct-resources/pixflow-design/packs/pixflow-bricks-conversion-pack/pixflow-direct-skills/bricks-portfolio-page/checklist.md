# Bricks Portfolio Page Checklist

## P0

- [ ] Work/project hierarchy is varied.
- [ ] At least one featured project or editorial highlight exists.
- [ ] No project results, client names, metrics, or awards are invented.
- [ ] Project cards have purposeful image or placeholder roles.
- [ ] The page includes a path to enquire about similar work.
- [ ] All generated classes are short BEM names.
- [ ] Pixflow variables are used for generated styling.

## P1

- [ ] Gallery is not a plain uniform grid unless requested.
- [ ] Project labels are useful and not filler.

## P2

- [ ] Final CTA references the portfolio context.
