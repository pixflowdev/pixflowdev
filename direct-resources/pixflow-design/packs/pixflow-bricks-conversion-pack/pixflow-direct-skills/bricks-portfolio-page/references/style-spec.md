# Bricks Portfolio Page Style Spec Contract

## Token Rules

- Project cards use `var(--foreground)`.
- Project media uses `var(--radius--surface)` and `var(--border--surface)`.
- Borders use `var(--border)`.
- CTAs use `var(--radius--btn)`.

## Layout Style Rules

- Prioritize image rhythm and editorial spacing.
- Use accent for project tags or CTA, not every card.
- Keep mobile project cards scannable and not over-tall.
