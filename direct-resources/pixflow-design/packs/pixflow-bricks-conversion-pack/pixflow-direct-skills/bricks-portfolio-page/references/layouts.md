# Bricks Portfolio Page Layout References

## Default Rhythm

1. `portfolio_magazine_hero` - positioning.
2. `portfolio_featured_case` - featured project.
3. `portfolio_bento_gallery` - varied project grid.
4. `portfolio_process_split` - approach.
5. `portfolio_proof_editorial` - proof.
6. `portfolio_contact_cta` - enquiry.

## Layout Rules

- Use one dominant case-study preview.
- Mix large and small project cards.
- Do not fabricate client outcomes.
