# Bricks Portfolio Page Element Contract

## Allowed Elements

- `section`, `container`, `block`, `div`
- `heading`, `text-basic`, `button`, `image`

## Structure

- Project cards use editable blocks with image or placeholder, project title, scope, and optional CTA.
- Feature projects use a content split or bento card, not just a text card.
- Contact CTA uses Bricks buttons and text elements.

## Bricks Rules

- Use BEM blocks such as `work`, `case`, `gallery`, `approach`, and `contact`.
- Do not output raw galleries, scripts, or unsupported embeds.
