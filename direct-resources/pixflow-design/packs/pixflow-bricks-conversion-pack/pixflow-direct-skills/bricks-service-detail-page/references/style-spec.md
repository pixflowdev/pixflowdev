# Bricks Service Detail Style Spec Contract

## Token Rules

- Section backgrounds use `var(--background)`.
- Panels, FAQ items, and proof blocks use `var(--foreground)`.
- Borders use `var(--border)` and visible dividers use `max(var(--border--btn), 1px)`.
- Cards and media use `var(--radius--surface)` and `var(--border--surface)`.
- Buttons use `var(--radius--btn)` and `var(--border--btn)`.

## Layout Style Rules

- Keep the design calm and service-specific.
- Use accent for CTA, step numbers, or one proof cue, not every heading.
- Avoid repeated equal cards across fit, benefits, and process.
