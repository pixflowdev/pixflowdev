# Bricks Wireframe Sketch Layout References

## Default Rhythm

1. `wireframe_hero_structure` - first viewport structure.
2. `wireframe_content_zones` - content or feature blocks.
3. `wireframe_split_section` - supporting copy/media zone.
4. `wireframe_process_flow` - sequence if needed.
5. `wireframe_objection_stack` - support/FAQ.
6. `wireframe_cta_block` - action close.

## Layout Rules

- Use labelled low-fidelity regions.
- Keep decoration minimal.
- Use varied structural patterns, not identical grey cards throughout.
