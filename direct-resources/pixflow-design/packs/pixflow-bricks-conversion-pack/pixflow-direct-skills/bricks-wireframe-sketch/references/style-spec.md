# Bricks Wireframe Sketch Style Spec Contract

## Token Rules

- Use `var(--background)`, `var(--foreground)`, `var(--border)`, and `var(--text)`.
- Keep borders visible with `max(var(--border--btn), 1px)`.
- Use `var(--radius--surface)` only as a structural container radius.

## Layout Style Rules

- Low fidelity means neutral surfaces, simple borders, and restrained typography.
- Avoid shadow, gradient, image, and product-polish treatments.
- Keep labels readable at mobile widths.
