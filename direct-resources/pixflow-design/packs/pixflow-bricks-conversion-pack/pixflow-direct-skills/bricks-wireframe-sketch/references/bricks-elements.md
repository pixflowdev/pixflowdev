# Bricks Wireframe Sketch Element Contract

## Allowed Elements

- `section`, `container`, `block`, `div`
- `heading`, `text-basic`, `button`

## Structure

- Use blocks for wireframe regions, cards, media placeholders, and action zones.
- Use text labels like `Hero media`, `Feature card`, or `CTA action` when real content is unknown.
- Do not use image elements unless the user explicitly supplied a wireframe image for reuse.

## Bricks Rules

- Use BEM blocks such as `wireframe`, `wireframe__zone`, `wireframe__card`, and section-specific BEM names.
- Keep responsive overrides simple and flat.
