# Bricks Screenshot Redesign Layout References

## Default Rhythm

1. `redesign_reference_hero` - first viewport rebuilt from reference cues.
2. `redesign_bento_section` - improved content/features.
3. `redesign_collage_split` - editable visual interpretation.
4. `redesign_proof_or_support` - supplied proof only.
5. `redesign_cta` - conversion close.

## Layout Rules

- Extract hierarchy, spacing, rhythm, palette intent, media roles, and UI structure.
- Do not paste screenshots as page sections.
- Do not copy inaccessible assets that were not supplied for reuse.
