# Bricks Screenshot Redesign Style Spec Contract

## Token Rules

- Map reference color/style cues into active Pixflow tokens.
- Use raw values only when the active route cannot express the style safely.
- Use `var(--background)`, `var(--foreground)`, `var(--border)`, `var(--radius--surface)`, and `var(--radius--btn)`.

## Layout Style Rules

- Preserve reference atmosphere without brittle pixel copying.
- Improve clarity and responsiveness where the reference is weak.
- Avoid generic SaaS blue defaults unless the reference or design system calls for them.
