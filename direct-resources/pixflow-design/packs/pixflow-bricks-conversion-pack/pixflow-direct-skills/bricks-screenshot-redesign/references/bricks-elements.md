# Bricks Screenshot Redesign Element Contract

## Allowed Elements

- `section`, `container`, `block`, `div`
- `heading`, `text-basic`, `button`, `image`

## Structure

- Rebuild reference layout as native Bricks sections and blocks.
- Use `image` only for user-supplied assets intended for reuse or explicit replacement media.
- Use editable blocks for UI screenshots, browser frames, feature cards, wireframes, and product previews.

## Bricks Rules

- Use BEM blocks based on the rebuilt section role, not the source screenshot filename.
- Do not expose reference-analysis notes in the final page.
