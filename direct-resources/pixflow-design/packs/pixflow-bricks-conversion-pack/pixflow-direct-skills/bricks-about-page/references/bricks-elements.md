# Bricks About Page Element Contract

## Allowed Elements

- `section`, `container`, `block`, `div`
- `heading`, `text-basic`, `button`, `image`

## Structure

- Use `block` for story panels, values, process steps, quote panels, and CTA bands.
- Use `image` only when the prompt supplies or requests founder, team, place, or process visuals.

## Bricks Rules

- Use BEM blocks such as `story`, `values`, `approach`, `proof`, and `cta`.
- Keep text editable in Bricks text/heading elements.
- Do not use raw HTML for timeline or quote structures.
