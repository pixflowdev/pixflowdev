# Bricks About Page Style Spec Contract

## Token Rules

- Use `var(--background)` for sections and `var(--foreground)` for story/proof surfaces.
- Use `var(--heading)` and `var(--text)` for text colors.
- Use `var(--border)`, `var(--radius--surface)`, and `var(--radius--btn)`.

## Layout Style Rules

- Give the page a more editorial rhythm than a service grid.
- Accent should support hierarchy through small cues, not dominate the page.
- Keep cards at or below 8px radius only if the active design system requires it; otherwise use the Pixflow surface token.
