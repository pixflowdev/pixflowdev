# Bricks About Page Layout References

## Default Rhythm

1. `about_magazine_hero` - point-of-view hero.
2. `about_story_split` - origin or approach story.
3. `about_values_bento` - varied principles.
4. `about_approach_timeline` - working method.
5. `about_proof_editorial` - proof or trust.
6. `about_cta_contrast` - next step.

## Layout Rules

- Use editorial hierarchy and measured spacing.
- Do not turn values, proof, and process into three identical card sections.
- Use honest placeholders when details are missing.
