# Bricks About Page Skill

Use this pack for story-led about pages. The page should communicate the origin, point of view, values, approach, trust cues, and next step without vague mission statements.

Do not invent founders, team members, awards, credentials, or dates. If information is missing, use honest labels and business-specific language from the prompt.
