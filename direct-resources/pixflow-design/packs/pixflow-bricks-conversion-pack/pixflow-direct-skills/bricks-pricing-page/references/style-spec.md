# Bricks Pricing Page Style Spec Contract

## Token Rules

- Cards use `var(--foreground)`.
- Borders use `var(--border)` and visible borders use `max(var(--border--btn), 1px)`.
- Buttons use `var(--radius--btn)` and `var(--border--btn)`.
- Featured card emphasis uses accent sparingly and remains token based.

## Layout Style Rules

- Make package differences easy to scan.
- Keep long feature lists compact on mobile.
- Do not rely on color alone to mark a recommended plan.
