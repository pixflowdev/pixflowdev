# Bricks Release Notes One-Pager Bricks Element Contract

Use Bricks-native editable elements only: section, container, block/div, heading, text, button/link, icon where appropriate, image only when real media exists, and simple form-like fields when the page type needs them.

Do not output raw HTML widgets, unsupported custom elements, iframe scripts, shortcodes, fake screenshot image IDs, or empty wrappers. UI, dashboard, phone, board, email, release, and documentation previews must be built from editable Bricks blocks, labels, cards, lines, chips, and panels.
