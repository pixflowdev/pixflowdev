# Bricks Release Notes One-Pager Skill

Use this pack for release notes, changelog, launch notes, and product update pages. The final output must remain editable Bricks JSON using Pixflow variables, semantic global classes, and short BEM names.

Before planning sections, identify the user's real task, the page state being represented, and the proof or support needed. Do not convert this into a generic marketing homepage. Use Pixflow Design-style critique to remove repeated card grids, fake claims, empty media wrappers, and weak hierarchy before Bricks compilation.
