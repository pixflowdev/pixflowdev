# Bricks Element Rules

- Use normal Bricks sections, containers, blocks, headings, text, buttons, and images.
- Use supplied image attachments through image/media slots when the section needs real product or screenshot media.
- For unavailable product imagery, use editable Bricks blocks as labelled product-media placeholders rather than unrelated stock photography.
- Keep class names page-scoped and short: `brand-hero`, `brand-hero__media`, `product-grid__card`, `flavour__tile`.
- Do not output HTML/CSS/JS as the final target; the compiler creates Bricks JSON.
