# Style Spec

- Use the active Pixflow Framework route as the final implementation authority.
- Product-brand pages may use stronger accent/contrast sections than SaaS pages, but colours must still be expressed through Pixflow token object shapes or supplied brand values mapped to tokens.
- Keep headings large and playful only where the brand supports it; body copy must remain readable.
- Avoid one-note palettes by using section-level colour changes: hero, flavour discovery, product grid, trust, and CTA can each have a deliberate role.
- Use rounded product cards, packaging media frames, pill badges, mood tiles, trust chips, and clear CTA buttons.
