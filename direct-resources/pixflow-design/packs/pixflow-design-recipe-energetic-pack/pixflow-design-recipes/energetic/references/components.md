# Energetic Component Rules

## Navigation

- Bold brand wordmark inside a thick-bordered box.
- Monospaced navigation links (e.g. `[ SHOP ]`, `[ STORY ]`, `[ JOIN ]`).
- Clean primary CTA button with a thick 4px border and a solid offset shadow.
- Mobile menu button uses a simple, bold triple-line burger icon with a thick border.

## Badges and Labels

- Technical or technical-adjacent monospaced labels using brackets or slashes (e.g., `[ BATCH // 04 ]` or `// VITAL ENERGY`).
- Thick 4px borders and a high-contrast background fill.
- Small font sizes (11px to 13px) but uppercase and letter-spaced to maintain high legibility.

## Buttons

- Primary button: thick 4px border, bright orange background (`#EA580B`), white uppercase text, and a solid offset shadow.
- Secondary button: thick 4px border, soft peach background (`#FFEDD5`), dark indigo text, and a solid offset shadow.
- State guidelines:
  - Hover: translation shift (`transform: translate(-2px, -2px)`) and a slightly larger shadow, creating a tactile "pop-up" effect.
  - Active: translation shift down (`transform: translate(2px, 2px)`) with the shadow size reduced, simulating a button press.
  - Focus-visible: thick outline to ensure high visibility.

## Product / Feature Cards

- Thick 4px borders on all sides with a solid offset shadow (`4px 4px 0px 0px`).
- Card layout: badge -> bold title -> monospaced specs -> description paragraph.
- Cards scale up on hover (`transform: scale(1.02)`) with a smooth transition.
- Avoid squeezing too many cards in a single row. Stick to a 2-column or 3-column layout.

## Form Elements

- Inputs: thick 4px borders, white background, large padding.
- Input focus state: border color changes to primary orange (`#EA580B`) or gains a solid offset outline.
- Labels: bold, uppercase, placed above the input fields.
- Submit buttons: styled identically to primary buttons with offset shadows.

## Motion Cues

- Hover scaling uses a snappy ease-out or spring-based curve.
- Avoid complex parallax or layout shifts. Focus on button state transitions and card scaling.
