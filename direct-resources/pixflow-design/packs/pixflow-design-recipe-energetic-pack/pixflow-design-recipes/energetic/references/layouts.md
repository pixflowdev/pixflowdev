# Energetic Layout Patterns

## Hero With Active Center

Purpose: establish the energetic style immediately in the first viewport.

- Full section with a warm peach background (`#FFEDD5`) or deep indigo background (`#1D1836`).
- Bold, centered or asymmetric structure.
- Small monospaced category badge or label (e.g. `[ ENERGY // VIBE ]`).
- Bold, uppercase display headline (e.g., in Limelight font), keeping the title large but readable.
- Centered high-fidelity product mockup (e.g. can, bottle, device) wrapped in a thick 4px border.
- Primary CTA button styled with a thick 4px border and a solid offset shadow.
- Concise, motivating product statement.

Desktop pattern:
- Centered container up to 1440px inside a 1600px desktop review viewport.
- Hero content area 500px to 700px tall.
- Avoid squeezing side-by-side elements. If the hero is a split layout, ensure both the product media column and the title/copy column are wide enough to breathe.

Mobile pattern:
- The navigation bar becomes compact.
- The hero stacks vertically: badge -> headline -> product mockup -> CTA.
- The product mockup scales down to avoid overflow.
- Headline uses responsive font scaling to avoid clipping.

## Structured 2x2 Grid

- Four feature cards arranged in a 2-column, 2-row layout.
- Each card has a thick 4px border, soft peach/orange background, and a solid offset shadow (`4px 4px 0px 0px`).
- Include bold titles, monospaced specs, and simple icons or badges.
- Avoid dense multi-column layouts that squeeze cards on desktop. A 2x2 layout is preferred to give each card enough width.

## Alternating Split

- Two columns: one features a thick-bordered, solid-shadowed image or illustration panel, and the other features active copy and bullet lists.
- Alternate the order (image/text, text/image) in subsequent sections.
- If the split layout causes text to wrap into skinny columns on narrow viewports, convert the layout to a single column or move body copy below the heading.

## Quote bubble / Testimonial Row

- Testimonials placed in thick-bordered cards with solid offset shadows.
- Add a monospaced attribution line (e.g., `// ATOMIC ATHLETICS`).
- Align cards side-by-side on desktop, stacking into a single column on mobile.

## Active Call to Action

- Large card or full-width band using the deep indigo background (`#1D1836`) to create contrast.
- Bold uppercase heading (e.g., "GET ACTIVE / SHOP NOW").
- Large input field with a thick 4px border (for email signups) and a heavy-bordered button with a solid offset shadow.

## Footer

- Heavy top border (solid 4px).
- Two to four columns of links using bold, monospaced headers.
- Restrained, uppercase brand wordmark.
