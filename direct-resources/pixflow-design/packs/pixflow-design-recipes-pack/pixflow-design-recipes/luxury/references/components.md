# Luxury Component Rules

## Navigation

- Minimal: wordmark, three to five links, one quiet action.
- Use low-density spacing and hairline bottom rule when needed.
- Hide non-critical links on mobile before the header wraps.

## Capsule Label

- Small uppercase text.
- Hairline border or soft accent fill.
- Do not make badges bright or playful.

## Button

- Primary: off-white fill, dark text, pill or soft rectangle.
- Secondary: transparent, hairline border, off-white text.
- Minimum height 44px.
- Focus-visible uses an accent outline outside the element.
- Hover should be subtle: small background shift, border shift, or opacity change.

## Image Frame

- Use disciplined crop and aspect ratio.
- Radius should be 0 to 24px depending on brand.
- A thin border or ring can add refinement.
- Do not use generic stock-like blurred backgrounds.

## Product / Collection Card

- Image first, text second.
- Short title, category, and one practical line.
- Optional metadata such as edition, material, or appointment.
- Avoid five-plus equal cards on one row.

## Divider / Rule

- Thin line only.
- Use for pacing and editorial structure.
- Avoid boxed grids everywhere.

## Forms

- If the brief needs capture, use one compact appointment/inquiry form.
- Labels must remain visible.
- Inputs use dark surface, ivory text, hairline border, and clear focus state.

## Motion

- Reveal elements with opacity and small translate.
- No bounce, spinning, or decorative hover games.
- Reduced motion must reveal everything immediately.
