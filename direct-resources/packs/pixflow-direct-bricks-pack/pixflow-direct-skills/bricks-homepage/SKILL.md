# Bricks Homepage Skill

Use this pack for complete homepages. The page should introduce the business, show proof early, explain the offer, vary the middle sections, and end with a clear next step.

Avoid generic homepage rhythm. Do not use the same centered section header and equal card grid repeatedly. Use the attached context, screenshot, or Pixflow design system to choose a clear visual direction before planning sections.
