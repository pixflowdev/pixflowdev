# Pixflow Direct Bricks Pack

This pack provides Bricks-specific generation references for Pixflow Direct Generation.

Contents:

- `pixflow-direct-skills`: Pixflow Direct skill files and Bricks-compatible section skeletons.
- `pixflow-direct-references`: Pixflow Direct Bricks references used by the compiler and planner.
- `bricks-builder-skills`: Bricks Builder skill documentation for model context.

This pack is documentation and reference data only. It does not include Bricks Builder source code and does not grant a Bricks Builder license.
