# Pixflow Direct Design Pack

This pack provides Open Design-inspired generation intelligence for Pixflow Direct Generation.

Contents:

- `open-design-skills`: Open Design skill files used as design and scenario references.
- `open-design-templates`: Open Design page/prototype template skills and lightweight text side files used for skill routing, layout libraries, and P0/P1/P2 checklists. Heavy binary media has been omitted from this pack.
- `open-design-craft`: craft guidance, anti-slop rules, accessibility, typography, color, and state coverage references.
- `open-design-prompts`: Open Design prompt-stack references including discovery, visual directions, panel behavior, and system prompts.

The Pixflow plugin uses this as optional reference material. It should not override the active site's Pixflow Framework tokens or design-system settings.
