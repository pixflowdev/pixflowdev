# Bricks Homepage Checklist

## P0

- [ ] The page introduces the business and offer clearly in the hero.
- [ ] Proof or trust appears early, not only near the end.
- [ ] No two adjacent sections use the same layout family.
- [ ] All generated classes are short BEM names.
- [ ] Pixflow variables are used in Bricks object shapes.

## P1

- [ ] The homepage has one dominant visual idea.
- [ ] Middle sections vary in density and hierarchy.

## P2

- [ ] Final CTA contrasts with the previous section.
